# Grocerlyst - e-Portfolio

## Project Description
A mobile application to help users manage groceries and track real-time shopping budgets.

## SDG Theme
* **SDG 12:** Responsible Consumption & Production

## Feature List
* **2 New Screens:** loginScreen and barcodeInspectorScreen
* **Web API:** Fetches real product details from OpenFoodFacts API
* **Hardware Sensor:** Uses Camera sensor to scan barcodes
* **Cloud Storage:** Uses Firebase Firestore to store shared community data remotely

## Setup Instructions
1. Clone the repository.
2. Open the project in Android Studio.
3. Sync Gradle and run the app on an Android device with camera access.
